# Product Discovery Agent

## Overview

Product Discovery Agent helps founders validate ideas before building. The agent analyzes problems, identifies target users, evaluates opportunities, and recommends MVP features.
<img width="3152" height="1137" alt="Product Discovery Agent" src="https://github.com/user-attachments/assets/1f93dd24-023c-4cc1-bc6d-219b50d2c439" />

## Problem

Product discovery often requires extensive research, competitor analysis, and idea validation. This process is repetitive and time-consuming.

## Solution

The Product Discovery Agent automates research and generates a structured discovery report that helps founders make informed decisions.

## Features

* Problem Analysis
* User Pain Point Identification
* Market Opportunity Assessment
* Competitor Insights
* MVP Recommendations
* Risk Assessment
* Success Metrics

## Workflow

Product Idea → Product Discovery Agent → Discovery Report → Founder

## Tech Stack

* n8n
* OpenAI / Grok
* Airtable
* Google Search

## Overview

NumoraIQ is a Career & Project Intelligence Platform designed to help aspiring data analysts, students, career switchers, and professionals manage projects, track skill development, monitor productivity, and measure career growth.

Unlike traditional project management tools, NumoraIQ focuses on the intersection of:

* Project Execution
* Career Development
* Portfolio Building
* Productivity Analytics
* AI-Powered Planning

The goal is to help users transform ideas into completed projects while building valuable professional skills and strengthening their portfolios.

## Problem

Many aspiring analysts and professionals struggle to:

* Keep track of multiple learning and portfolio projects
* Measure progress toward career goals
* Understand which projects provide the greatest skill growth
* Organize milestones, deadlines, and deliverables
* Prioritize projects based on career impact
* Create structured project plans

Existing project management tools focus on tasks rather than personal and professional growth.

---

## Solution

NumoraIQ combines project tracking, analytics, portfolio management, and AI planning into a single platform.

Users can:

* Manage projects and milestones
* Track productivity and completion trends
* Measure skill development
* Assess portfolio value
* Generate AI-powered project roadmaps
* Monitor career growth over time

---

## Features

### Dashboard

* Total Projects
* Active Projects
* Completed Projects
* At-Risk Projects
* Career Growth Score
* Portfolio Strength Score
* Productivity Metrics

### Project Management

* Create and manage projects
* Track project status and progress
* Organize milestones
* Store project resources
* Add notes and documentation
* Track estimated vs actual effort

### Career Impact Score

Each project contributes to a Career Impact Score based on:

* SQL Skill Growth
* Power BI Skill Growth
* Python Skill Growth
* Business Understanding
* Portfolio Value

This helps users focus on projects that provide the greatest professional return.

### Analytics Dashboard

Interactive insights including:

* Projects by Category
* Projects by Status
* Completion Trends
* Skill Development Trends
* Productivity Metrics
* Hours Invested

### AI Project Planner

Generate complete project roadmaps from a simple idea.

Example:

Input:

Build a customer churn dashboard

Output:

* Project Summary
* Milestones
* Timeline
* Deliverables
* Risk Analysis
* Skill Development Assessment
* Career Impact Assessment

### Project Detail Workspace

Each project includes:

* Overview
* Milestones
* Notes
* Resources
* Time Tracking

## Why I Built This

As a Data analyst, I wanted a tool that went beyond task management and focused on tracking meaningful progress toward career goals.

NumoraIQ was created to connect project execution with skill development, portfolio growth, and professional advancement.

The platform serves as both a productivity tool and a career development companion.
